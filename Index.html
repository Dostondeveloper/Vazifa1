JavaScript nima ?

JavaScript (talaffuzi: /ˈdʒɑːvəskrɪpt/; qisqartmasi: JS /ˈdʒeɪ.ɛs./) yuqori pogʻonali dasturlash tilidir. 1995-yilda amerikalik dasturchi Brendan Eich tomonidan yaratilgan. JavaScript tili barcha asosiy brauzerlar tomonidan ishlatiladi. JavaScript orqali biz o'zimiz yaratgan saytlarmizdi ishlatamiz oddiy misol qilib odamnin olsak Css bilan tashqi ko'rnishni stil berib olsak js bilan uni harakatga keltiramiz.
